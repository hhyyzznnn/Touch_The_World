# 한국 중소기업 SMS/알림톡 발송 서비스 가이드

## 🎯 한국 중소기업에서 가장 많이 사용하는 서비스

### 1순위: 카카오 알림톡 (국내 대행사 사용) ⭐⭐⭐

**한국 시장 점유율**: 매우 높음
- 일간 발송량 1억 건 이상
- 2022년 기준 2017년 대비 5배 이상 증가
- 중소기업 대부분이 사용

**주요 대행사:**
1. **비즈엠** (https://www.bizmsg.kr/)
   - 업계 최저가 8원/건
   - 가입 즉시 알림톡 100건 무료
   - 1,300개 고객사 실적
   - 24시간 장애 모니터링

2. **솔라피(SOLAPI)** (https://solapi.dev/)
   - SMS, 카카오톡, RCS, 네이버 톡톡 통합
   - 해외 발송 지원
   - 커머스 자동화 기능

3. **비즈뿌리오** (https://www.bizppurio.com/)
   - 알림톡, 친구톡, SMS, RCS 통합
   - 실시간 스팸 모니터링
   - 웹 발송 및 시스템 연동

4. **프로디지** (https://kakaobiz.kr/)
   - 국내 최저가 API 8원
   - 알림톡 실패 시 자동 문자 전환
   - 전문 컨설팅 제공

**장점:**
- ✅ 비용 매우 저렴 (8-20원/건)
- ✅ SMS 대비 50% 이상 비용 절감
- ✅ 설정 간단 (대행사가 대부분 처리)
- ✅ 한국어 지원 완벽
- ✅ 사용자 친화적 (카카오톡으로 수신)
- ✅ 보안성 우수 (피싱/스미싱 차단)

**소요 시간**: 30분-1시간 (대행사 사용 시)

### 2순위: 카카오 알림톡 (직접 API 연동)

**난이도**: ⭐⭐⭐ (복잡함)
**소요 시간**: 2-3일 (심사 대기 포함)

**장점:**
- ✅ 비용 저렴 (8-20원/건)
- ✅ 완전한 제어 가능

**단점:**

- ❌ 비즈니스 채널 개설 필요
- ❌ 템플릿 등록 및 심사 필요
