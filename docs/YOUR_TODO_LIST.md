## 📋 해야 할 작업 (비즈엠 사용 시)

### 1단계: 비즈엠 가입 및 설정 (20분)
- [ ] [비즈엠 웹사이트](https://www.bizmsg.kr/) 접속
- [ ] 회원가입 (무료)
- [ ] 카카오 비즈니스 채널 연동
  - 이미 만든 비즈니스 채널 연결
  - 발신 프로필 키 확인 및 복사
- [ ] 대시보드에서 발신 프로필 키 확인

### 2단계: 인증번호 템플릿 등록 (10분)
- [ ] 비즈엠 대시보드 → 템플릿 관리
- [ ] 새 템플릿 작성
- [ ] 템플릿 내용 입력:
  ```
  [터치더월드] 휴대폰 인증
  
  인증번호: #{인증번호}
  
  위 인증번호를 입력해주세요.
  (유효시간: 5분)
  ```
- [ ] 변수 설정: `#{인증번호}` 추가
- [ ] 템플릿 제출 (비즈엠이 심사 대행)
- [ ] 템플릿 코드 확인 및 복사

### 3단계: API 키 발급 (5분)
- [ ] 비즈엠 대시보드 → API 설정
- [ ] API Key 발급 및 복사
- [ ] API Secret 발급 및 복사

### 4단계: 환경 변수 설정 (5분)

#### 로컬 개발 환경 (.env 파일)
```env
# 비즈엠 API 설정
BIZMSG_API_KEY="발급받은-API-Key"
BIZMSG_API_SECRET="발급받은-API-Secret"
BIZMSG_SENDER_KEY="발신-프로필-키"
BIZMSG_TEMPLATE_CODE="인증번호-템플릿-코드"
```

- [ ] `.env` 파일에 위 내용 추가
- [ ] 값들이 올바르게 입력되었는지 확인

#### Vercel 프로덕션 환경
- [ ] [Vercel Dashboard](https://vercel.com/dashboard) 접속
- [ ] 프로젝트 선택 → **Settings** → **Environment Variables**
- [ ] 다음 환경 변수 추가:
  - [ ] `BIZMSG_API_KEY`: 3단계에서 복사한 API Key
  - [ ] `BIZMSG_API_SECRET`: 3단계에서 복사한 API Secret
  - [ ] `BIZMSG_SENDER_KEY`: 1단계에서 복사한 발신 프로필 키
  - [ ] `BIZMSG_TEMPLATE_CODE`: 2단계에서 확인한 템플릿 코드
- [ ] 각 환경 변수 저장 확인

### 5단계: 코드 수정 (필요 시)
- [ ] 비즈엠 API 연동 코드 추가
- [ ] 또는 비즈엠이 제공하는 SDK 사용
- [ ] 테스트

## 📋 해야 할 작업 (직접 API 연동 시)

### 1단계: 템플릿 등록 및 심사 (1-2일)
- [ ] 카카오 비즈니스 센터 접속
- [ ] 알림톡 템플릿 작성
- [ ] 템플릿 내용:
  ```
  [터치더월드] 휴대폰 인증
  
  인증번호: #{인증번호}
  
  위 인증번호를 입력해주세요.
  (유효시간: 5분)
  ```
- [ ] 변수 설정: `#{인증번호}` 추가
- [ ] 템플릿 제출
- [ ] 카카오 심사 대기 (1-2일)
- [ ] 템플릿 코드(template_code) 확인 및 복사

### 2단계: 카카오 개발자 센터 설정 (20분)
- [ ] [카카오 개발자 센터](https://developers.kakao.com/) 접속
- [ ] 내 애플리케이션 만들기
- [ ] 앱 이름: "Touch The World"
- [ ] 플랫폼 설정: 웹 플랫폼 추가
- [ ] 제품 설정 → 카카오 i Connect Message 활성화
- [ ] 서비스 계약 확인 (디케이테크인과 계약 필요)
- [ ] Client ID 발급 및 복사
- [ ] Client Secret 발급 및 복사

### 3단계: 환경 변수 설정 (5분)

#### 로컬 개발 환경 (.env 파일)
```env
# 카카오 비즈니스 메시지 API (직접 연동)
KAKAO_BM_CLIENT_ID="발급받은-Client-ID"
KAKAO_BM_CLIENT_SECRET="발급받은-Client-Secret"
KAKAO_BM_SENDER_KEY="발신-프로필-키"
KAKAO_BM_VERIFICATION_TEMPLATE_CODE="인증번호-템플릿-코드"
KAKAO_BM_BASE_URL="https://bizmsg-web.kakaoenterprise.com"
```

- [ ] `.env` 파일에 위 내용 추가
- [ ] 값들이 올바르게 입력되었는지 확인

#### Vercel 프로덕션 환경
- [ ] [Vercel Dashboard](https://vercel.com/dashboard) 접속
- [ ] 프로젝트 선택 → **Settings** → **Environment Variables**
- [ ] 다음 환경 변수 추가:
  - [ ] `KAKAO_BM_CLIENT_ID`: 2단계에서 복사한 Client ID
  - [ ] `KAKAO_BM_CLIENT_SECRET`: 2단계에서 복사한 Client Secret
  - [ ] `KAKAO_BM_SENDER_KEY`: 비즈니스 채널의 발신 프로필 키
  - [ ] `KAKAO_BM_VERIFICATION_TEMPLATE_CODE`: 1단계에서 확인한 템플릿 코드
  - [ ] `KAKAO_BM_BASE_URL`: `https://bizmsg-web.kakaoenterprise.com` (기본값)
- [ ] 각 환경 변수 저장 확인

### 4단계: 코드 확인
- [ ] 코드는 이미 구현되어 있음 (`lib/kakao-alimtalk.ts`)
- [ ] 환경 변수만 설정하면 자동으로 사용됨
- [ ] 테스트
