## 📋 대행사 사용 시 해야 할 작업

### 방법 A: 비즈엠 사용 (추천) ⭐

#### 1단계: 비즈엠 가입 및 설정 (20분)
- [ ] [비즈엠 웹사이트](https://www.bizmsg.kr/) 접속
- [ ] 회원가입 (무료)
- [ ] 카카오 비즈니스 채널 연동 (이미 있으므로 연결만)
- [ ] 발신 프로필 키 확인

#### 2단계: 인증번호 템플릿 등록 (10분)
- [ ] 비즈엠 대시보드에서 템플릿 작성
- [ ] 템플릿 내용:
  ```
  [터치더월드] 휴대폰 인증
  
  인증번호: #{인증번호}
  
  위 인증번호를 입력해주세요.
  (유효시간: 5분)
  ```
- [ ] 변수 설정: `#{인증번호}` 추가
- [ ] 템플릿 제출 (비즈엠이 심사 대행)

#### 3단계: API 키 발급 (5분)
- [ ] 비즈엠 대시보드 → API 설정
- [ ] API Key 발급 및 복사
- [ ] API Secret 발급 및 복사

#### 4단계: 환경 변수 설정 (5분)
```env
# 비즈엠 API 설정
BIZMSG_API_KEY="발급받은-API-Key"
BIZMSG_API_SECRET="발급받은-API-Secret"
BIZMSG_SENDER_KEY="발신-프로필-키"
BIZMSG_TEMPLATE_CODE="인증번호-템플릿-코드"
```

#### 5단계: 코드 수정 (필요 시)
- [ ] 비즈엠 API 연동 코드 추가 (현재는 카카오 직접 API 사용 중)
- [ ] 또는 비즈엠이 제공하는 SDK 사용

**총 소요 시간**: 약 30분-1시간

---

### 방법 B: 솔라피 사용

#### 1단계: 솔라피 가입 및 설정 (20분)
- [ ] [솔라피 웹사이트](https://solapi.dev/) 접속
- [ ] 회원가입 (300포인트 무료체험)
- [ ] 카카오 비즈니스 채널 연동
- [ ] 발신 프로필 키 확인

#### 2단계: 인증번호 템플릿 등록 (10분)
- [ ] 솔라피 대시보드에서 템플릿 작성
- [ ] 템플릿 내용 동일하게 작성
- [ ] 변수 설정: `#{인증번호}` 추가
- [ ] 템플릿 제출

#### 3단계: API 키 발급 (5분)
- [ ] 솔라피 대시보드 → API 설정
- [ ] API Key 발급 및 복사
- [ ] API Secret 발급 및 복사

#### 4단계: 환경 변수 설정 (5분)
```env
# 솔라피 API 설정
SOLAPI_API_KEY="발급받은-API-Key"
SOLAPI_API_SECRET="발급받은-API-Secret"
SOLAPI_SENDER_KEY="발신-프로필-키"
SOLAPI_TEMPLATE_CODE="인증번호-템플릿-코드"
```

**총 소요 시간**: 약 30분-1시간

## 📋 직접 API 연동 시 해야 할 작업 (복잡함)

- [ ] 알림톡 템플릿 작성

  ```

  위 인증번호를 입력해주세요.
  ```

- [ ] 변수 설정: `#{인증번호}` 추가
- [ ] 템플릿 제출
- [ ] 카카오 심사 대기 (1-2일)
- [ ] 템플릿 코드(template_code) 확인

### 2단계: 카카오 개발자 센터 설정 (20분)
- [ ] [카카오 개발자 센터](https://developers.kakao.com/) 접속
- [ ] 내 애플리케이션 만들기
- [ ] 앱 이름: "Touch The World"
- [ ] 플랫폼 설정: 웹 플랫폼 추가
- [ ] 제품 설정 → 카카오 i Connect Message 활성화
- [ ] 서비스 계약 확인 (디케이테크인과 계약 필요)
- [ ] Client ID 발급 및 복사
- [ ] Client Secret 발급 및 복사

### 3단계: 환경 변수 설정 (5분)
```env
# 카카오 비즈니스 메시지 API (직접 연동)
KAKAO_BM_CLIENT_ID="발급받은-Client-ID"
KAKAO_BM_CLIENT_SECRET="발급받은-Client-Secret"
KAKAO_BM_SENDER_KEY="발신-프로필-키"
KAKAO_BM_VERIFICATION_TEMPLATE_CODE="인증번호-템플릿-코드"
KAKAO_BM_BASE_URL="https://bizmsg-web.kakaoenterprise.com"
```

### 4단계: 코드 확인
- [ ] 코드는 이미 구현되어 있음 (`lib/kakao-alimtalk.ts`)
- [ ] 환경 변수만 설정하면 자동으로 사용됨

**총 소요 시간**: 약 2-3일 (심사 대기 포함)
