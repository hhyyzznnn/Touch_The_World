# SMS 인증번호 발송 서비스 비교

## 📊 난이도 비교

### 1. Twilio (해외 서비스)
**난이도**: ⭐ (가장 쉬움)

**장점:**
- ✅ 설정이 매우 간단 (계정 생성 → API 키 발급)
- ✅ 문서가 명확하고 국제 표준
- ✅ 무료 체험 크레딧 제공
- ✅ 즉시 사용 가능
- ✅ 한국 번호 지원 (추가 설정 필요)

**단점:**
- ❌ 해외 서비스 (한국 번호는 별도 설정)
- ❌ 한국어 지원 제한적
- ❌ 비용이 상대적으로 높을 수 있음
- ❌ 한국 중소기업에서 덜 사용됨

**소요 시간**: 약 15-20분

**한국 시장 점유율**: 낮음 (주로 글로벌 서비스나 해외 서비스에 적합)

---

### 2. 카카오 알림톡 (국내 서비스) ⭐ 추천
**난이도**: ⭐⭐ (중간 - 국내 대행사 사용 시 간단)

**장점:**
- ✅ **한국에서 가장 널리 사용됨** (일간 1억 건 이상 발송)
- ✅ 국내 서비스 (한국어 지원 완벽)
- ✅ 카카오톡으로 발송 (사용자 친화적)
- ✅ 템플릿 기반 (일관된 메시지)
- ✅ **비용이 매우 저렴** (8-20원/건)
- ✅ **국내 대행사 사용 시 설정 간단** (비즈엠, 솔라피 등)
- ✅ 보안성 우수 (피싱/스미싱 차단)
- ✅ SMS 대비 50% 이상 비용 절감

**단점:**
- ❌ 직접 API 연동 시 설정 복잡 (OAuth 2.0, 템플릿 심사)
- ❌ 템플릿 등록 및 심사 필요 (1-3일 소요)
- ❌ 비즈니스 채널 개설 필요

**소요 시간**: 
- 직접 API 연동: 약 2-3일 (심사 대기 포함)
- **국내 대행사 사용: 약 30분-1시간** ⭐

**한국 시장 점유율**: 매우 높음 (중소기업 대부분 사용)

## 🔧 구현 난이도 상세 비교

### Twilio 구현 단계
1. ✅ [Twilio 가입](https://www.twilio.com/try-twilio) (5분)
2. ✅ 대시보드에서 Account SID, Auth Token 확인 (2분)
3. ✅ 전화번호 발급 (무료 체험 번호 사용 가능) (3분)
4. ✅ 환경 변수 설정 (2분)
5. ✅ 코드 테스트 (3분)

**총 소요 시간**: 약 15분

---

### 카카오 알림톡 구현 단계
1. ⚠️ [카카오 비즈니스 채널 개설](https://bizmsg.kakao.com/) (30분)
   - 사업자 등록증 필요
   - 채널 개설 신청
   - 승인 대기 (1-2일)
2. ⚠️ 템플릿 등록 및 심사 (1-2일)
   - 인증번호 발송 템플릿 작성
   - 카카오 심사 제출
   - 승인 대기
3. ⚠️ API 키 발급 (20분)
   - 카카오 개발자 센터에서 앱 등록
   - Client ID, Client Secret 발급
4. ⚠️ OAuth 2.0 인증 구현 (30분)
   - 액세스 토큰 발급 로직
   - 토큰 갱신 로직
5. ⚠️ 알림톡 발송 API 연동 (1시간)
   - API 엔드포인트 호출
   - 에러 처리
6. ⚠️ 테스트 (30분)

**총 소요 시간**: 약 2-3일 (심사 대기 포함)

## 💰 비용 비교

### Twilio
- 무료 체험: $15 크레딧 제공
- 한국 SMS: 약 $0.05-0.10/건
- 월 구독료: 없음 (사용량 기반)

### 카카오 알림톡
- 기본 요금: 월 구독료 + 건당 요금
- 건당 요금: 약 ₩20-30/건 (서비스에 따라 다름)
- 템플릿 등록: 무료
