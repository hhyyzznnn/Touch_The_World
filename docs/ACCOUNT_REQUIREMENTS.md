## 🔵 구글 SNS 로그인

### 개인 계정 사용 가능 ✅

**결론**: 개인 Google 계정으로 사용 가능합니다.

**이유:**
- Google Cloud Console은 개인 계정으로도 프로젝트 생성 가능
- OAuth 클라이언트 ID/Secret 발급 가능
- "외부" 앱 타입으로 설정 시 개인 계정 사용 가능
- 회사 계정이 필수 아님

**주의사항:**
- OAuth 동의 화면에서 "외부" 앱 타입 선택
- 테스트 모드에서는 테스트 사용자만 사용 가능
- 프로덕션 배포 시 Google 검토 제출 필요 (개인 계정도 가능)

**설정 방법:**
1. 개인 Google 계정으로 [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 새 프로젝트 생성
3. OAuth 동의 화면 설정 (외부 앱 타입)
4. OAuth 클라이언트 ID 생성
5. 환경 변수 설정

**참고:**
- 개인 계정으로 생성한 프로젝트는 "[개인계정]"으로 표시됨
- 기능적으로는 회사 계정과 동일하게 작동

## 🟡 카카오 알림톡

### 개인 계정 사용 불가능 ❌

**결론**: 비즈니스 채널 개설 시 **사업자 등록증 필수**입니다.

**이유:**
- 카카오 알림톡은 "비즈니스 채널"이 필요함
- 비즈니스 채널 개설 시 사업자 등록증(또는 고유번호증) 제출 필수
- 개인 계정으로는 비즈니스 채널 개설 불가능

**대안:**
1. **대행사 사용** (비즈엠, 솔라피 등)
   - 대행사가 비즈니스 채널을 대행해주는 경우도 있음
   - 하지만 여전히 사업자 등록증이 필요할 수 있음
   - 대행사마다 정책이 다르므로 확인 필요

2. **일반 채널 사용** (제한적)
   - 개인 계정으로 일반 채널 개설 가능
   - 하지만 알림톡 발송은 비즈니스 채널이 필요
   - 일반 채널로는 알림톡 발송 불가능

**필수 요구사항:**
- ✅ 사업자 등록증 (또는 고유번호증)
- ✅ 비즈니스 채널 개설
- ✅ 템플릿 등록 및 심사

## 💡 추천 방법

- 개인 계정으로는 불가능

**대안:**
1. 회사 사업자 등록증으로 비즈니스 채널 개설
2. 대행사 문의 (비즈엠, 솔라피 등) - 개인 계정 사용 가능 여부 확인
3. 일단 구글 로그인만 구현하고, 카카오 알림톡은 나중에 진행
