## 🔵 Google Cloud Console (구글 OAuth)

### 결제 계정 요구사항

**개인 계정**: ✅ 가능
- 개인 Google 계정으로 프로젝트 생성 가능
- 결제 계정도 개인 계정으로 설정 가능
- 자동 결제 (신용카드, 직불카드, 은행 계좌)
- 명세서(Statement) 발급

**회사 계정**: ⚠️ 권장
- 회사 Google 계정으로 프로젝트 생성 가능
- 인보이스 계정 설정 가능 (후불 결제)
- 월별 인보이스 발급
- 결제 기간 설정 가능 (예: 만 30일)
- 회사 회계 처리 용이

**권장사항:**
- 초기 개발/테스트: 개인 계정으로 가능
- 프로덕션 배포: 회사 계정 권장 (인보이스 발급, 회계 처리)

- 사업자등록증 필수
- 대표번호 필수
- 회사 계정으로만 가입 가능

**중요:**
- 카카오 비즈니스 채널이 회사 계정으로 이미 있다면
- 비즈엠 계정도 **회사 계정으로 가입**해야 함
- 회사 비즈니스 채널과 연동하려면 회사 계정 필요
