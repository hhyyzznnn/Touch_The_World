# 데이터 현황 및 비어있는 부분

## 📊 현재 데이터 현황

### ✅ 데이터 있음
- **사업 실적 (Achievements)**: 시드 스크립트로 일부 데이터 존재 (`scripts/seed-achievements.ts`)

### ❌ 데이터 없음 (비어있는 부분)

#### 1. 프로그램 데이터
- **프로그램 (Program)**: 없음
- **프로그램 이미지 (ProgramImage)**: 없음
- **프로그램 일정 (ProgramSchedule)**: 없음
- **영향**: 
  - 홈페이지 "프로그램 유형" 카테고리 클릭 시 빈 페이지
  - `/programs` 페이지에 "등록된 프로그램이 없습니다" 표시
  - 프로그램 상세 페이지 접근 불가

#### 2. 행사 데이터
- **행사 (Event)**: 없음
- **행사 이미지 (EventImage)**: 없음
- **영향**:
  - 홈페이지 "최근 진행 행사" 섹션 빈 상태
  - `/events` 페이지에 "등록된 행사가 없습니다" 표시
  - 행사 상세 페이지 접근 불가

#### 3. 학교 데이터
- **학교 (School)**: 없음
- **영향**:
  - 행사 페이지의 학교 필터 비활성화
  - 학교별 페이지 (`/school/[id]`) 접근 불가
  - 행사 등록 시 학교 선택 불가

#### 4. 자료실 데이터
- **문서 (Document)**: 없음
- **영향**:
  - `/documents` 페이지에 "등록된 자료가 없습니다" 표시
  - 자료 다운로드 불가

#### 5. 문의 데이터
- **문의 (Inquiry)**: 없음 (사용자가 직접 생성 가능)
- **영향**:
  - 관리자 대시보드에서 "대기 중인 문의: 0" 표시
  - 문의 관리 페이지 빈 상태

## 🔧 해결 방법

### 방법 1: 시드 스크립트 실행 (권장)

전체 샘플 데이터를 한 번에 추가할 수 있는 시드 스크립트를 실행하세요:

```bash
npm run db:seed
```

이 스크립트는 다음 데이터를 생성합니다:
- 학교 4개
- 프로그램 6개 (각 카테고리별)
- 행사 6개
- 자료실 문서 4개
- 문의 2개

### 방법 2: 관리자 페이지에서 수동 추가

1. 개발 서버 실행: `npm run dev`
2. 관리자 페이지 접속: `http://localhost:3000/admin`
3. 각 섹션에서 데이터 추가:
   - 프로그램 관리 → 새 프로그램 추가
   - 행사 관리 → 새 행사 추가
   - 학교 관리 → 새 학교 추가
   - 자료실 관리 → 새 자료 추가

### 방법 3: 사업 실적만 추가

사업 실적 데이터만 추가하려면:

```bash
npm run db:seed:achievements
```

## 📝 시드 스크립트 상세

### `scripts/seed-all.ts`
전체 샘플 데이터를 생성하는 통합 시드 스크립트입니다.

**생성되는 데이터:**
- **학교**: 서울중앙초등학교, 강남중학교, 서울고등학교, 부산국제중학교
- **프로그램**: 
  - 제주도 수학여행 3박 4일 (수학여행)
  - 부산 수련활동 2박 3일 (수련활동)
  - 경주 역사 문화 탐방 (현장 체험학습)
  - 일본 교류 프로그램 (해외탐방)
  - AI 융합 교육 프로그램 (교육연수)
  - 진로 탐색 특강 (진로특강)
- **행사**: 6개 (다양한 학교와 프로그램 조합)
- **자료실**: 안전 매뉴얼, 공문 템플릿, 안내문 양식, 교육과정 계획서
- **문의**: 2개 (대기 중, 완료)

### `scripts/seed-achievements.ts`
사업 실적 데이터만 생성하는 시드 스크립트입니다.

## ⚠️ 주의사항

1. **기존 데이터 덮어쓰기**: 시드 스크립트를 여러 번 실행하면 중복 데이터가 생성될 수 있습니다.
2. **이미지 URL**: 시드 스크립트의 이미지 URL은 Unsplash 플레이스홀더입니다. 실제 이미지로 교체해야 합니다.
3. **파일 URL**: 자료실 문서의 파일 URL은 예시입니다. 실제 파일 업로드 후 URL로 교체해야 합니다.
